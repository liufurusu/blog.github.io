Git Bash中继续输入！

git clone https://git.coding.net/itbulu/gitstudy.git(我们自己的仓库路径)
在Git Bash中继续输入：

git init
git add .
git commit -m “version 3.9”
git remote add origin https://coding.net//<项目名称>.git
git push origin master
